Django blog development introductory tutorial: Preface

Django blog tutorial 2017年5月11日 Dreamers 4 comments 588 read

Django is an open source web framework written in Python that can be used to quickly build a high-performance website.

Django makes it easier to build better Web apps more quickly and with less code.

Django allows you to build better web applications faster, with less code, and easier.

This tutorial will help you quickly develop your own blog website using Django.

Tutorial Features

Free, Chinese, zero-based, complete project, based on the latest version of Django 1.10 and Python 3.5.

Take you step by step to develop your own blog website from scratch, helping you master Django development skills as quickly as possible.

Who is suitable for this tutorial

This tutorial is mainly for newcomers to Django who have zero basic knowledge.

You only need a little basic knowledge of the Python language to read this tutorial smoothly.

If you already have some experience in Django development, you can also learn more Django development skills from this tutorial.
